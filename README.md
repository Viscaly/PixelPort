# PixelPort

**PixelPort** is a modern e-commerce platform built with **Blazor** and **.NET 8.0**, designed for tech enthusiasts. It offers a wide range of products including gaming consoles, accessories, software, and more. The application integrates **PostgreSQL** for robust data management and provides an interactive, responsive user experience.

## 🚀 Features

- **Interactive UI:** Built with Blazor Server using interactive components for real-time updates.
- **Product Management:** Browse, search, and filter products including consoles, games, and tech gadgets.
- **Responsive Design:** Fully responsive layout for desktop, tablet, and mobile devices.
- **Admin Portal:** Secure admin login to manage products and inventory.
- **Database:** PostgreSQL backend for scalable and reliable data storage.
- **Modern Tech Stack:** Leveraging .NET 8.0 and the latest C# features.

## 💻 Tech Stack

| Layer      | Technology                 |
|-----------|----------------------------|
| Frontend   | Blazor Server (.NET 8.0)  |
| Backend    | ASP.NET Core               |
| Database   | PostgreSQL                 |
| Styling    | CSS3, Flexbox, Grid        |
| Hosting    | Any ASP.NET-compatible server |
